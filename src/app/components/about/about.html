About Me:

I am a passionate and dedicated BCA student with a strong interest in web development and modern frontend technologies. This mini blog application is one of my hands-on projects built using Angular, where I focused on understanding core concepts such as routing, component-based architecture, and state management. Through this project, I explored how to create a smooth single-page application that allows users to navigate between pages, manage content, and interact with data without full page reloads. The application includes features like creating, editing, and deleting posts, along with basic authentication to simulate real-world user access control. In addition to functionality, I have also focused on designing a clean and user-friendly interface that reflects modern UI trends. Each blog post is structured with meaningful details such as title, author, publishing date, and reading time, making the experience more realistic and engaging. I have used localStorage to persist data, ensuring that user-created content remains available even after refreshing the application. This project has helped me strengthen my understanding of Angular fundamentals while also improving my problem-solving and debugging skills. I am continuously learning and exploring new technologies to enhance my development skills. My goal is to build scalable and efficient web applications and eventually work on real-world projects that create meaningful user experiences. This blog app is a small step in my journey, showcasing both my technical abilities and my commitment to learning and growth in the field of software development.
